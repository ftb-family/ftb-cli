### 🏠 [首页](https://github.com/ftb-family/ftb-cli#readme)

## 安装

```sh
npm install ftb-cli -g
```

## 使用

```sh
 ftb init [name] // vue or react
```

## 本地开发调试
```js 
git clone https://github.com/ftb-family/ftb-cli.git

cd ftb-cli

npm install

cd packages/ftb-cli

npm link

ftb init react
```
