> A cli tool for create vue and react application.

### 🏠 [Homepage](https://github.com/ftb-family/ftb-cli#readme)

## Install

```sh
npm install ftb-cli -g
```

## Usage

```sh
 ftb init [name] // vue or react
```
> Currently, React cli is ready, Vue will supporting later.

## Run in local
```js
git clone [project]

cd project

npm install

npm link

ftb init react
```
